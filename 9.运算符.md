# 运算符

## 常用的运算符

1. 算数运算符：+、-、*、/、%、++、--

2. 赋值运算符：==

3. 关系运算符：>、<、>=、<=、==、!= instanceof <>

4. 逻辑运算符：&&、||、!

5. 位运算符：&、|、^（异或全部为0取0，有一个为1取1，都为1取0）、~（取反）、>>（除以2）、<<（乘以2）、>>>

6. 条件运算符：?:

   ``` java
   x ? y : z 如果x为真，则x=y，否则x=z
   ```

7. 扩展赋值运算符：+=、-=、*=、/=

## 幂运算

2的3次方

``` Java
double pow = Math.pow(3,2);
```
