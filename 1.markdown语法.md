# markdown语法

## 标题

1. 一个#加空格是一级标题
2. 两个#加空格是二级标题，依次类推，最多六级标题

## 字体

1. **粗体** 字体的两边都加上两个*

2. *斜体* 字体的两边都加上一个*

3. ***斜体加粗*** 字体两个都加上三个*

4. ~~删除线~~ 字体两边都加上两个~

## 引用

> 引用，字体左边加上>

## 分割线

1. 三个-

---

2. 三个*

***

## 图片插入

1. !+[图片名称]+(图片路径)，路径可以是本地的也可以是网上的

![图片](D:\photo\1.jpg)

## 超链接

[连接名称]+(链接URL)

[Security菜鸟](https://www.cnblogs.com/SecurityHacker/)

## 列表

1. 有序列表。序号+.空格
2. 无序列表。-加空格，+加空格

## 表格

1. 手写。如下，然后在源代码中将空行删除即可

列名1|列名2|列名3
--|--|--|
元素1|元素2|元素3

2. 直接插入表格

   |      |      |      |
   | ---- | ---- | ---- |
   |      |      |      |

## 代码

1. 行内代码。一个反引号（TAB键旁边的）+空格。 ` public`

2. 代码块。三个反引号加空格

``` python
print("Hello World!")
```
